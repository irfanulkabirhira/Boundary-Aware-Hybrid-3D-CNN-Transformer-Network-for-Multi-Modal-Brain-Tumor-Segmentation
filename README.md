# Boundary-Aware-Hybrid-3D-CNN-Transformer-Network-for-Multi-Modal-Brain-Tumor-Segmentation
Hybrid 3D CNN–Transformer for brain tumor segmentation on BraTS 2021. Combines attention fusion, transformer bottleneck &amp; boundary-aware loss. Achieves Dice 0.8742, Recall 0.9379. Includes ablation study, SHAP explainability &amp; full reproducible pipeline.
